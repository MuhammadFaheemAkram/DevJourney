# Changelog

All notable changes to DevJourney will be documented in this file.

The project follows a phase-based build history while it is being prepared as a portfolio-quality open-source sample.

## Unreleased

- Continue expanding integration, Room, Flow, and Compose UI test coverage.
- Add screenshots and visual regression guidance.
- Improve dashboard aggregation with repository-backed state.

## 0.7.0 - Phase 7

- Added open-source documentation.
- Added contribution, conduct, security, changelog, and license files.
- Added GitHub issue templates, pull request template, and Android CI workflow.
- Finalized README for portfolio use.

## 0.6.0 - Phase 6

- Added repository-backed analytics screen.
- Added DataStore-backed settings.
- Added WorkManager reminder scheduling.
- Added analytics and settings use cases and tests.

## 0.5.0 - Phase 5

- Added challenges, resources, and global search features.
- Added challenge completion, resource bookmarking, and search use cases.
- Added unit tests for Phase 5 behavior.

## 0.4.0 - Phase 4

- Added notes CRUD and goals progress tracking.
- Added notes and goals use cases and unit tests.

## 0.3.0 - Phase 3

- Added roadmaps, roadmap detail, topic detail, progress tracking, and topic bookmarking.

## 0.2.0 - Phase 2

- Added Room, DAOs, fake API, repository implementations, mappers, and demo data seeding.

## 0.1.0 - Phase 1

- Added Compose foundation, Hilt bootstrap, Material 3 theme, navigation shell, drawer, bottom navigation, and placeholder screens.
